mycommon.msauth module#

class mycommon.msauth.PublicClientAuthentication(username, client_id=None, tenant_id=None, scope=[], password=None, passwordEnv=None, cacheToken=True, cache_style=<class 'mycommon.msauth.LocalSharePointCredsCache'>)#

Bases: Authentication

Initialize sharepoint filesystem as normal filesystem. Note: Current module supports 2 types of authentication 1. Username and Password 2. Interactive using device code flow. This method is generally required when user login is backed with MFA, MSAL PublicAuth

Parameters

  • username (str) – username, this will be used for authentication/cache refresh

  • client_id (str) – Azure Active Directory Application Client Id

  • tenant_id (str) – Azure Active Directory Application Client Id

  • scope (list, optional) – Active Directory Application access scopes, by default []

  • password (str, optional) – Password for the username, required only if authentication is done by username and password, by default None

  • passwordEnv (str, optional) – Environment variable which contains the password. passwordEnv takes precedence over the password parameter, by default None

  • cacheToken (bool, optional) – Do you want to enable cache, helps to reduce the use of credentials everytime user logs in, by default True

  • cache_style (Callable[[str],Union[dict,None]], optional) – Method to fetch/write into cache using given username. LocalSharePointCredsCache is created for you which will use user home directory to store credential which can be overridden using SHAREPOINT_CREDENTIALS_CACHE_FILE environment variable. Note: method should take username as parameter and implement read & write function for cache. Sample can be found below , by default LocalSharePointCredsCache

 1class LocalSharePointCredsCache:
 2"""Simple local file based credential cache implementation.
 3
 4Parameters
 5----------
 6username : str
 7    Logged in username
 8"""
 9def __init__(self,username:str) -> None:
10    """
11
12    Parameters
13    ----------
14    username : str
15        Logged in username
16    """
17    self.username=username
18    default_credentials_location=os.path.expanduser("~")
19    default_credentials_with_file=os.path.join(default_credentials_location,'.spfs_creds.json')
20    cache_creds_file=os.environ.get('SHAREPOINT_CREDENTIALS_CACHE_FILE',default_credentials_with_file)
21    cache_dir,self.cache_creds_file=os.path.split(cache_creds_file)
22    self.osfs=OSFileSystem(cache_dir)
23    if self.osfs.is_file(self.cache_creds_file):
24        try:
25            spfs_creds=self.osfs.read_file_content(self.cache_creds_file)['Body'].read() #.decode('utf-8')
26            self.spfs_creds=json.loads(spfs_creds)
27        except Exception as e:
28            self.spfs_creds={}
29    else:
30        self.spfs_creds={}
31
32def read(self)-> dict:
33    """Get user credentials cache
34
35    Returns
36    -------
37    dict
38        dict containing token, refresh_token and expiry timestamp
39    """
40    try:
41        return self.spfs_creds[self.username]
42    except Exception as e:
43        return {}
44
45def write(self,values:dict)->None:
46    """Updates credentials cache
47
48    Parameters
49    ----------
50    values : dict
51        dict containing updated cache details. Schema to be same as response of read() method.
52    """
53    self.spfs_creds[self.username]=values
54    self.osfs.write_file_content(self.cache_creds_file, io.BytesIO(json.dumps(self.spfs_creds).encode('utf-8')),chunk_write=False)
