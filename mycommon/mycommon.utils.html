Copies data from one source_fs to other as streaming data. This method is primarly useful when the object size is large and to conserve system memory.

Parameters

  • object_path (str) – Name of the object that needs to be copied or path of the object name, if the source_fs path is different.

  • source_fs (FileSystem) – Should be a subclass of FileSystem object

  • target_fs (FileSystem) – Should be a subclass of FileSystem object

  • target_fs_object_path (Optional[str], optional) –

    Absolute path of target_fs or relative path of initialised target_fs.

    Can also be used if you want to rename the file in the target_fs If no value is given, then it will consider the given source object_path, by default None

  • read_byte_size (int, optional) – Default 5MB chunks will be read from source_fs. Note: Feel free to increase the byte_size that needs to read as per the source_fs capability , by default 4**10*5

  • show_progress (bool, optional) – , by default True

  • write_byte_size (Union[int, str], optional) – Leave to default which can handle by type for Filesystem, by default ‘default’

  • source_fs_extra_args (dict, optional) – Source file system related extra args

  • target_fs_extra_args (dict, optional) – Target file system related extra args
